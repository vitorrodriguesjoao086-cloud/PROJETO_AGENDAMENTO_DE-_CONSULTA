# MedAgenda - Sistema de Agendamento Medico

Projeto da disciplina de Programacao.

## Como rodar

```bash
# Instalar dependencias
python -m pip install -r requirements.txt

# Criar dados de exemplo
python seed_data.py

# Rodar o servidor
python manage.py runserver
```
